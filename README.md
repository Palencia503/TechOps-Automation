# Automatización
## Proyecto de Automatización (PAC-7)
Este repositorio contiene los scripts de automatización desarrollados durante las 5 semanas. El objetivo es optimizar la gestión de sistemas mediante Bash (Ubuntu) y PowerShell (Windows).

📂 Estructura del Repositorio
El proyecto está organizado por entornos y semanas para facilitar la localización de los scripts:

🐧 Bash (Ubuntu)

Semana 1 / InfoSistema: Script para extraer información básica del hardware.

Semana 2 / Usuarios: Automatización del alta masiva de usuarios desde un archivo .txt.

Semana 3 / Backups: Script de copia de seguridad local con registro de logs y automatización mediante Cron(opcional lo de crontab).

Semana 4 / Monitorizacion: Script de control de procesos críticos y generación de informes de estado.

🪟 PowerShell (Windows)

Semana 1 / InfoSistema: Obtención de datos del sistema mediante cmdlets nativos.

Semana 2 / Usuarios: Creación masiva de cuentas locales gestionando permisos de administrador y políticas de ejecución.
